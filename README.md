# Student Task Manager

A command-line task management application built with Python to help students organize and manage their academic tasks.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Save tasks using JSON
* Automatically load saved tasks when the program starts
* Basic exception handling for invalid input and file errors

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON
* File Handling
* Exception Handling
